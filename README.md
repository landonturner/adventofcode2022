## Using Day Generation Script

1. grab your session from network tab in browser
2. save your session in `.envrc.local`:

```
export AOC_SESSION=1234abcd
```

3. run `./new_day.sh 3` to generate day 3 and download input

_note: sessions are valid for 30 days so you should be able to use the same one for the whole month_
